## 📋 **使用示例**

### 🌟 **场景1: 新用户快速开始**
```bash
# 1. 启动BSC配置向导
zkstack bsc wizard

# 2. 或者使用快速启动
zkstack bsc quick-start --name my-bsc-project

# 3. 进入项目目录
cd my-bsc-project

# 4. 初始化生态系统
zkstack ecosystem init

# 5. 启动服务器
zkstack server run
```
